# 🐦 Tappy Bird

A simple and addictive tap-to-fly mobile game built with Unity.

## 🎮 About

Tappy Bird is a Flappy Bird-style game where you control a bird through obstacles. Tap to fly, collect coins, and unlock new birds!

**Features:**
- Tap to fly with smooth physics
- Collect coins by passing obstacles
- Unlock 5 different birds
- High score tracking
- Dynamic difficulty as you progress

## 🎯 How to Play

1. **Tap/Click** to make the bird fly upward
2. **Avoid** hitting the pipes, ground, or ceiling
3. **Collect** coins for passing obstacles
4. **Unlock** new birds with your coins
5. **Beat** your highest score!

## 🛠️ Built With

- Unity Engine
- C#
- DOTween (animations)
- Unity Input System
